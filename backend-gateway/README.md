# Backend Gateway - Marketing & SEO Intelligence

API Gateway desarrollado con Express + TypeScript

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La API estará disponible en `http://localhost:4000`

## Build

```bash
npm run build
npm start
```

## Variables de Entorno

Copia `.env.example` a `.env` y configura las variables necesarias.

## Estructura

- `src/routes/` - Rutas de la API
- `src/controllers/` - Controladores
- `src/middlewares/` - Middlewares (auth, roles)
- `src/services/` - Servicios (comunicación con FastAPI)
- `src/utils/` - Utilidades (JWT, etc.)
